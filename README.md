# 🔴⚪🔵 Guerreiras da Fonte
 
> Landing page fictícia da primeira torcida organizada dedicada ao time feminino do Esporte Clube Bahia.
 
---
 
## 📋 Sobre o projeto
 
As **Guerreiras da Fonte** é uma torcida organizada fictícia criada como projeto acadêmico, com o objetivo de representar e apoiar o futebol feminino do Esporte Clube Bahia — o Esquadrão de Aço.
 
O site foi desenvolvido com foco em identidade visual forte, utilizando as cores oficiais do Tricolor Baiano e inspirado no site oficial do clube.
 
---
 
## 🖥️ Tecnologias utilizadas
 
- HTML5
- CSS3 (Flexbox, Custom Properties, Gradientes, Animações)
- Google Fonts (Archivo Black, Inter, Roboto)
 
---
 
## 📄 Seções do site
 
- **Nav** — Logo, slogan, links de navegação e botão "Faça Parte"
- **Notícias** — Cards com as últimas notícias do time feminino
- **Nossa História** — Origem e missão da torcida
- **Mulheres de Aço** — Apresentação do elenco feminino
- **Footer** — Links de navegação e informações gerais

## 🚀 Como rodar
 
1. Clone o repositório:
```bash
git clone https://github.com/Erickhitman/Guerreiras-da-Fonte.git
```
2. Abra o arquivo `index.html` no navegador.
 
> Não requer instalação de dependências — o projeto é 100% HTML e CSS puro.
